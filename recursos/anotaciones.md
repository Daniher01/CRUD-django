# Anotaciones del tutorial

#### Project Setup

- activar entorno virtual
- instalar django
- iniciar proyecto
    ``` shell
        django-admin startproject nombre_proyecto . 
        #el . para que lo inizalice en la misma carpeta
    ```
- correr el servidor   
    ``` shell
        python manage.py runserver
    ```
- iniciar una App
    ``` shell
        python manage.py startapp nombre_app
    ```
